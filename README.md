# Credlytic

Credlytic is an AI-powered payment tracker that automatically reads transaction messages to log expenses and credits in real time. It supports multiple bank accounts and wallets, manual entry, and visual analytics to help users understand their financial behavior.

## Features
- Automatic transaction detection from SMS
- Debit and credit classification
- Manual transaction entry
- Spending and savings analytics
- Multi-account support
